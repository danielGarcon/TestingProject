# WebdriverIO & Artillery Project

## Overview
Automated browser tests using WebdriverIO (POM, modular utilities, multi-browser) and load testing with Artillery.

## Setup
1. `npm install`
2. Configure browsers in `wdio.conf.js` as needed.

## Running Tests
- Run all tests: `npm run test`
- Run load test: `npm run loadtest`

## File Structure
- `src/`: Utilities, page objects, reusable test logic
- `test/`: Main test runner, specs
- `loadtest/`: Artillery config
- `wdio.conf.js`: WDIO config
- `package.json`: Scripts & dependencies
